# 功能说明
## 使用说明

### 1. 地图浏览
- 打开页面后，地图会自动定位到大运大学城区域
- 地图上显示周边餐厅的位置标记（橙色圆点）
- 点击标记可查看餐厅详情

### 2. 切换学校
- 顶部导航栏提供学校切换按钮
- 点击不同学校可筛选显示对应学校附近的餐厅

### 3. AI智能推荐
点击左上角「想怎么吃？」按钮，进入智能推荐模式：

#### 选择标签
- 系统随机展示7个标签供选择
- 多选支持：可同时选择多个标签（交集筛选）
- 换一换：点击可刷新标签列表，已选标签会保留

#### 可选标签说明（表格形式展示各类标签）

#### 推荐结果
- 系统根据所选标签的交集进行筛选
- 无匹配结果时，会随机推荐4家餐厅
- 点击推荐卡片可在地图上定位餐厅
- 点击「返回重新选择」可重新选择标签

### 4. 右侧餐厅列表
- 显示当前筛选/推荐的餐厅列表
- 点击列表项可在地图上定位并查看详情

### 什么叫针对学生群体的本地化美食指南？
学生的口味、用餐需求、预算等多方面因素都非常多样，但普遍而言希望是追求口味好，预算合适（当作食堂之外的选择），拍照好看，周围可能有一些其他玩的东西更好。

港中深附近这片区域是深圳国际大学园（大运大学城），可以算大学城，所以可以拿这里举例子。

而本地化更多地指？多大算本地？
那就按照深圳国际大学园的面积来算吧。东至龙飞大道，西至武深高速，南至沈海高速（机荷高速），北至如意路，2.66平方千米。但太小了，肯定不够。

MVP是什么？
1. 可以浏览的带地图的网页
	1. 可以看到餐厅/美食信息（来源于真实的平台，可跳转到对应平台买券或点外卖）
	2. 有路线规划，符合本地化线下用餐的需求。
2. AI辅助决策，给出多个标准化词语，方便用户找到自己想吃的/不想吃的东西。
